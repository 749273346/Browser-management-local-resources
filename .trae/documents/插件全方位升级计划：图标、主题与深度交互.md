# 插件全方位升级计划：广铁特色与深度交互 (最终版)

根据您的深度定制需求，我们将对插件进行**文化级**的重塑。这不仅仅是一个文件管理器，更是一个体现**广铁精神、铁路文化与时代浪潮**的专属应用。我们将把铁路元素、广铁配色以及积极向上的时代精神深度融入到 UI/UX 设计中。

## 1. 广铁特色视觉体系 (CR-Guangzhou Visual Identity)

### 1.1 专属主题定制 (Railway Themes)
我们将不再使用通用的科技主题，而是定制具有鲜明铁路特色的主题：
*   🚄 **“高铁时代” (CR High-Speed) - 默认主题**
    *   **配色**: 以**“复兴号”红/银**或**“和谐号”白/蓝**为主色调。
    *   **风格**: 现代、高速、流线型。背景可采用极简的高铁线条纹理，体现“时代浪潮”与“速度感”。
*   🛤️ **“铁轨坚守” (Iron Spirit)**
    *   **配色**: **深铁灰**与**路徽蓝**（铁路蓝）。
    *   **风格**: 稳重、坚毅。体现铁路职工扎实苦干的“铁路精神”。
*   🚂 **“历史长河” (Railway History)**
    *   **配色**: **复古黄铜色**与**蒸汽黑**。
    *   **风格**: 怀旧、文化底蕴。使用老式蒸汽机车、老站台元素作为装饰，致敬历史发展进程。

### 1.2 铁路元素图标设计
*   **App Icon**: 设计一个结合**铁路路徽**与**文件夹**元素的图标，或者以抽象的高铁车头为灵感。
*   **功能图标**:
    *   文件夹图标：不再是普通的黄色文件夹，可以设计带有**路徽印章**的档案袋样式。
    *   加载动画：使用**车轮滚动**或**信号灯变换**的微动效。
    *   Home 键：设计成**火车站**或**机务段**的图标。

### 1.3 界面细节与文化植入
*   **背景纹理**: 在侧边栏或空白区域，隐约展示**广铁辖区路线图**、**钢轨截面**或**接触网**的几何纹理。
*   **文案微调**:
    *   “Start Managing” -> **“发车 / 启程”**
    *   “Settings” -> **“调度中心”**
    *   “History” -> **“行车记录”**

## 2. 核心功能：智能管理与“忽略”系统

### 2.1 “忽略” (Ignore/Hide) 功能
*   **痛点解决**: 隐藏无需关注的系统文件，保持“站场”整洁。
*   **操作**: 右键 -> **“入库隐藏”**（将文件移入不可见列表）。
*   **管理**: 在“调度中心”中查看“停运/隐藏”的文件列表。

### 2.2 增删查改 (CRUD)
*   **右键菜单**: 提供重命名、新建、删除等基础功能。
*   **搜索**: 顶栏搜索框，支持快速查找“物资”（文件）。

## 3. 实施路线图

### 第一阶段：广铁视觉重塑
1.  **图标设计**: 制作铁路特色 App Icon 和内部功能图标。
2.  **主题引擎**: 实现“高铁时代”、“铁轨坚守”等主题配色方案。
3.  **UI 重构**: 将侧边栏、背景图、按钮样式全面“铁路化”。

### 第二阶段：核心 CRUD 与忽略功能
1.  **忽略逻辑**: 实现前端隐藏逻辑与持久化存储。
2.  **API 对接**: 完善后端文件操作接口。
3.  **交互实现**: 右键菜单组件开发。

### 第三阶段：效率与小工具
1.  **侧边栏**: 集成“行车日记”（笔记）和“重点物资”（收藏）。

我们将以**“高铁时代”**为首发主题，优先实现视觉重塑和忽略功能，让这款插件成为广铁职工得心应手的数字化工具。
