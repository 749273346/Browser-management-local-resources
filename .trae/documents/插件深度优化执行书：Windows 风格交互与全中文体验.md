收到您的反馈，我们将立即进行针对性调整。本计划将聚焦于**全中文本地化**、**Windows 11 风格右键菜单**以及**背景设置流程优化**，打造符合您使用习惯的专业工具。

## 1. 全中文界面本地化 (Localization)

### 1.1 移除英文对照
*   **执行内容**: 遍历所有界面组件，移除所有括号内的英文说明，仅保留专业、简洁的中文术语。
    *   `Appearance` -> **外观设置**

    *   `Tweaks` -> **视觉微调**
    *   `Welcome` -> **欢迎使用调度中心**

## 2. Windows 11 风格右键菜单 (Context Menu)

### 2.1 交互设计
完全复刻 Windows 11 的右键操作逻辑，取代浏览器默认菜单。

*   **选中文件时** (点击右键):
    *   📂 **打开** (Open)
    *   ✏️ **重命名** (Rename) - *弹出输入框修改名称*
    *   🗑️ **删除** (Delete) - *二次确认弹窗*
    *   👁️ **停运/隐藏** (Ignore) - *将文件加入忽略列表*
*   **选中空白处时** (点击右键):
    *   ➕ **新建文件夹** (New Folder)
    *   🔄 **刷新** (Refresh)

### 2.2 视觉风格
*   采用 **Acrylic (亚克力)** 半透明材质效果。
*   圆角矩形设计，悬停高亮，左侧带精美图标（使用 Lucide 图标库模拟 Windows 图标）。

## 3. 核心功能增强 (Backend API)

为了支持上述右键菜单，后端服务 (`server/index.js`) 将新增以下接口：
*   `POST /api/mkdir`: 创建新文件夹。
*   `POST /api/rename`: 重命名文件或文件夹。
*   `POST /api/delete`: 删除文件或文件夹 (物理删除)。

## 4. 设置中心体验优化

### 4.1 背景图设置流程
解决“不知道点哪里保存”的问题，优化交互逻辑：
*   **新增“确认应用”按钮**: 上传图片后，显示预览图，并出现明显的**“确认更换”**按钮，点击后才正式保存并生效。
*   **明确的上传区域**: 优化上传框样式，使其更像一个可点击的按钮。
