# 浏览器本地资源管理插件 - 用户操作手册

本手册将指导您完成软件的安装、配置和日常使用。

## 第一章：软件安装

### 1.1 获取安装包
请确保您已获得 `LocalResourceManager_Installer_v1.0.zip` 安装包。

### 1.2 开始安装
1.  解压安装包到任意位置。
2.  双击运行解压目录下的 **`Setup.bat`**。
3.  在弹出的安装向导中：
    *   **选择安装路径**：建议保持默认（`C:\Program Files\LocalResourceManager`），或点击 "Browse" 选择其他位置。
    *   点击 **"Install"** 按钮开始安装。
4.  等待安装进度条完成。安装程序会自动为您配置运行环境和开机自启服务。

### 1.3 安装浏览器插件
安装程序完成后，会自动打开浏览器扩展页面和插件文件夹。如果未自动打开，请手动操作：

#### Google Chrome 用户：
1.  在浏览器地址栏输入 `chrome://extensions` 并回车。
2.  开启右上角的 **"开发者模式"** 开关。
3.  点击左上角的 **"加载已解压的扩展程序"**。
4.  在弹出的窗口中，导航到您的安装目录（例如 `C:\Program Files\LocalResourceManager`），选中 **`extension`** 文件夹，点击“选择文件夹”。

#### Microsoft Edge 用户：
1.  在浏览器地址栏输入 `edge://extensions` 并回车。
2.  开启左侧菜单底部的 **"开发人员模式"** 开关。
3.  点击 **"加载解压缩的扩展"**。
4.  在弹出的窗口中，导航到您的安装目录，选中 **`extension`** 文件夹，点击“选择文件夹”。

## 第二章：日常使用

### 2.1 验证安装
1.  打开浏览器的一个**新标签页**。
2.  您应该能看到全新的本地资源管理界面，且左下角**没有**出现红色报错提示。
3.  如果显示文件列表，说明安装成功。

### 2.2 使用功能
*   **浏览文件**：点击文件夹图标即可进入下级目录。
*   **返回上级**：点击顶部的路径导航条（面包屑）可快速返回上级目录。
*   **打开文件**：
    *   支持预览的文件（如图片、文本）可直接在浏览器查看。
    *   其他文件点击后会调用系统默认程序打开。

## 第三章：常见问题 (FAQ)

**Q1: 打开新标签页提示“无法连接到后台服务”？**
*   **原因**：后台服务未启动或被拦截。
*   **解决**：
    1.  请尝试刷新页面。
    2.  检查电脑右下角是否有安全软件弹窗拦截了 Node.js 进程，请选择“允许”。
    3.  进入安装目录，双击运行 `start-background.bat` 手动启动服务。

**Q2: 电脑重启后插件还能用吗？**
*   **答**：可以。软件已配置开机自启，重启后服务会自动在后台运行，您无需任何操作。

**Q3: 如何卸载软件？**
*   **答**：直接删除安装目录，并在浏览器扩展页面移除该插件即可。
