# 浏览器本地资源管理插件 - 软件说明书

## 1. 软件简介
**浏览器本地资源管理插件 (Local Resource Manager)** 是一款专为提升工作效率设计的工具，旨在解决浏览器与本地文件系统之间的交互壁垒。通过本插件，用户可以在浏览器新标签页中直接、安全地管理本地文件资源，实现“所见即所得”的本地资源导航体验。

本软件采用轻量化架构：
1.  **后台服务 (Server)**：基于 Node.js，负责本地文件的安全读取和操作。支持绿色部署，自动处理运行环境。
2.  **前端界面 (Extension)**：基于 Chrome 扩展技术，提供现代化、美观的用户交互界面。

## 2. 核心功能
*   **新标签页集成**：无缝接管浏览器新标签页，打造高效的工作台入口。
*   **本地文件浏览**：支持可视化查看本地硬盘目录，提供网格/列表视图切换。
*   **智能颜色标记**：
    *   **一级分类**：支持对根目录下一级文件夹进行颜色标记（右键菜单）。
    *   **自动继承**：子文件夹自动继承父级颜色，确保视觉风格统一，避免杂乱。
*   **个性化壁纸**：
    *   支持自定义上传背景图片。
    *   支持**右键删除**已添加的壁纸缩略图，管理更加便捷。
*   **绿色便携**：无需安装，解压即用。内置环境检测脚本，支持无网环境下的自动配置。

## 3. 系统需求
*   **操作系统**：Windows 7 / 10 / 11 (64位推荐)
*   **浏览器**：Google Chrome, Microsoft Edge 或其他 Chromium 内核浏览器
*   **运行环境**：Node.js (脚本会自动检测并安装，无需预装)

## 4. 安全性说明
*   **本地运行**：所有数据处理均在用户本地电脑完成，**绝不上传**任何文件到互联网。
*   **权限最小化**：后台服务仅响应来自本插件的请求，并严格限制文件操作权限。
