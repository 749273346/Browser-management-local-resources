# 浏览器本地资源管理插件 - 软件说明书

## 1. 软件简介
**浏览器本地资源管理插件 (Local Resource Manager)** 是一款专为提升工作效率设计的工具，旨在解决浏览器与本地文件系统之间的交互壁垒。通过本插件，用户可以在浏览器新标签页中直接、安全地管理本地文件资源，实现“所见即所得”的本地资源导航体验。

本软件包含两个核心组件：
1.  **后台服务程序 (Server)**：运行在本地的高性能文件服务，负责安全地读取和管理文件。
2.  **浏览器插件 (Extension)**：基于 Chrome 内核的扩展程序，提供美观、现代化的用户界面。

## 2. 核心功能
*   **本地文件浏览**：直接在浏览器中查看本地硬盘的目录结构和文件。
*   **新标签页集成**：接管浏览器新标签页，每次打开新窗口都能快速访问常用资源。
*   **可视化管理**：提供现代化的文件图标、列表/网格视图切换。
*   **智能导航**：支持面包屑导航、常用路径收藏（开发中）。
*   **开机自启**：后台服务全自动运行，无需人工干预。

## 3. 系统需求
*   **操作系统**：Windows 7 / 10 / 11 (64位推荐)
*   **浏览器**：Google Chrome, Microsoft Edge 或其他 Chromium 内核浏览器
*   **硬件要求**：
    *   CPU: 1GHz 及以上
    *   内存: 512MB 及以上
    *   硬盘: 至少 200MB 可用空间

## 4. 安全性说明
*   **本地运行**：所有数据处理均在用户本地电脑完成，**绝不上传**任何文件到互联网。
*   **开源透明**：核心代码基于开源技术构建，无恶意后门。
*   **权限控制**：后台服务仅响应来自本插件的请求，确保数据安全。
